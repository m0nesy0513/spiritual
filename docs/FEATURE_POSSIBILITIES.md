## 🧭 分类 A：情绪即时干预（11 条）

### A1. 紧急平复按钮
首页常驻一个"我现在很焦虑"按钮，点击后直接进入 5 分钟快速平复流程：呼吸引导 → 一句安抚语 → 一个简单动作建议。不做分析，不做记录，只求快。
> [ ]

### A2. 呼吸练习计时器
内置 4-7-8 呼吸法、盒式呼吸、腹式呼吸三种模式的动画计时器，可设置 1/3/5 分钟。振动或音效提示。
> [ ]

### A3. 焦虑温度计
每次上传分析前先让用户滑一个 1-10 的焦虑值（已有采集），把这个数据做成独立入口——不传截图也能随手记一个"现在 7 分，原因：______"，形成情绪散点图。
> [ ]

### A4. 担忧书写（CBT 思维记录表）
结构化表单：情境 → 自动负性思维 → 支持证据 → 反对证据 → 替代思维 → 重评焦虑值。做完后存为笔记，可选投喂给 AI 做进一步分析。
> [ ]

### A5. 五分钟冥想引导
纯文字引导页，不需要音频。每 30 秒自动切换到下一句引导语，配合极简动画（呼吸球放大缩小）。完成后可记录感受。
> [ ]

### A6. 情绪急救包
用户预设一套对自己管用的东西——一首歌、一段话、一张照片、一个联系人——焦虑发作时一键打开，不需要在 app 里到处找。
> [ ]

### A7. 环境音混音器
提供 4-5 种循环环境音（雨声、篝火、溪流、白噪音、咖啡馆），支持同时播放并独立调音量。离线缓存音频文件。
> [ ]

### A8. 身体扫描引导
从脚趾到头部的逐部位放松文字引导，每个部位停留 20 秒，完成后生成一张"身体紧张热力图"（只是一个简单的自评标记，不需要传感器）。
> [ ]

### A9. 情绪日记
每天睡前推送一条提醒："今天让你最焦虑的一件事是什么？最感激的一件事是什么？" 两句话日记，自动归档时间线。
> [ ]

### A10. 替代行为建议库
当 AI 分析出焦虑触发场景后，从知识库中匹配对应的"替代行为"——比如"被老板批评后"推荐"去楼下走 5 分钟"—并支持用户自己添加替代行为。
> [ ]

### A11. 自我对话提示卡
预设一套基于 CBT 的自我提问卡："这个担心的最坏结果是什么？它发生的概率有多大？如果朋友遇到同样的事我会怎么劝他？"用户可以翻卡并写回答。
> [ ]

## 📈 分类 B：数据洞察与趋势（8 条）

### B1. 焦虑趋势周报
每周一自动生成一条推送摘要：本周上传次数、平均焦虑值、最高/最低点、最常出现的知识标签、相比上周的变化。可分享图片卡片。
> [ ]

### B2. 触发场景排行榜
基于历史分析数据，自动统计触发焦虑的 Top 10 场景（按频次 + 焦虑值加权），用柱状图展示。点击每个场景可查看相关历史记录。
> [ ]

### B3. 情绪日历热力图
以 GitHub 贡献图的形式展示过去一年的每日焦虑均值，颜色越深越焦虑。一眼看出"黑色月份"和好转趋势。
> [ ]

### B4. 应对效果追踪
每次分析后 AI 会推荐知识条目，用户后续是否再次因类似场景焦虑？横向对比：看了知识条目的场景 vs 没看的场景，焦虑值是否下降。
> [ ]

### B5. 标签时间线
按"工作焦虑""亲密关系""社交恐惧"等标签维度，分别画出焦虑值随时间的变化曲线。可同时勾选多个标签对比。
> [ ]

### B6. 焦虑时段分析
统计一天中哪个时段（早上/下午/晚上/深夜）焦虑感最频繁、最强烈。配合建议：比如"你的焦虑峰值在晚上 10 点，建议睡前 1 小时做呼吸练习"。
> [ ]

### B7. 知识条目疗效评分
用户看完百宝箱条目后可以点"有帮助/没帮助"，系统统计每条的帮助率。首页推荐时优先推荐高帮助率条目。
> [ ]

### B8. 年度心理报告
每年 12 月生成一份精美的年度报告：总分析次数、焦虑值轨迹、最常用的应对方法、最有帮助的一条知识、关键词云。可保存为图片分享。
> [ ]

## 🎮 分类 C：激励与游戏化（7 条）

### C1. 连续打卡系统
每日完成至少一次上传/笔记/情绪记录即计为打卡，连续 7/21/66/100/365 天分别解锁不同等级徽章。漏卡不归零但连续中断。
> [ ]

### C2. 经验值与等级
每次使用核心功能（上传分析、写笔记、看知识条目、做呼吸练习）获得经验值，升级解锁新主题色或新头像框。纯装饰，不锁功能。
> [ ]

### C3. 徽章收集册
设计 30-50 枚成就徽章，如"首次上传""连续 7 天""看了 50 条知识""焦虑值下降 3 分""反馈被采纳""安抚了 100 次自己"。已获得/未获得双态展示。
> [ ]

### C4. 每日挑战
每天推送一个小任务："今天记录一件让你平静的事""今天用一次呼吸练习""今天看一条百宝箱"。完成 3 个每日挑战获得当日奖励标记。
> [ ]

### C5. 应对技能树
把常用应对策略（深呼吸、写下来、转移注意力、找人聊聊、运动、正念）设计成一棵技能树，每次使用对应策略就加熟练度。熟练度满了解锁该策略的"进阶版"内容。
> [ ]

### C6. 里程碑时间线
自动检测并展示用户的里程碑节点：首次使用、焦虑值首次下降、连续打卡突破、知识条目阅读量破百……以时间线卡片呈现，可回顾。
> [ ]

### C7. 好友擂台（可选关闭）
匿名排行榜：本周打卡天数、本月分析次数等。仅展示自己在好友中的大致位置（如前 30%），不暴露具体数值。可随时关闭排名可见性。
> [ ]

## 🔗 分类 D：社交与连接（6 条）

### D1. 匿名互助信
用户可以写一封匿名信描述自己的困扰，AI 匹配一条相关知识条目 + 一段鼓励语回复。其他用户也可以写回信（同样匿名），每封信最多 3 封回信。
> [ ]

### D2. 鼓励传递
用户可以发送一条预设鼓励语（彩虹卡风格），随机分发给另一位用户。收到鼓励的用户可以看到一条温暖的话 + "来自某位同伴"。不建立双向联系。
> [ ]

### D3. 主题小屋
创建临时的匿名主题讨论空间（如"试用期焦虑""分手后重建"），24 小时后自动关闭。每个用户限发 3 条消息。不保留聊天记录。
> [ ]

### D4. 陪伴宣言
用户可以选择公开一个"陪伴状态"：比如"我正在应对社交焦虑，第 30 天"。其他用户可以看到有多少人在相似状态中，但无法看到具体是谁。纯粹的数字陪伴感。
> [ ]

### D5. 知识条目共读
显示"此刻有 X 人正在阅读这条知识"，以及阅读后的匿名感受标签云（"有用""被打动了""试过了"）。不做评论区，保持轻量。
> [ ]

### D6. 感谢信
用户可以给开发团队写一封感谢信或功能建议（区别于反馈系统的 bug/feature request），纯粹的情感表达。团队可以精选展示（经用户同意）。
> [ ]

## 🧰 分类 E：工具与实用功能（8 条）

### E1. 截图隐私模糊
上传截图前自动检测手机号/身份证号/真实姓名等敏感信息，提供一键模糊覆盖（纯前端处理，不传服务器）。保护用户隐私。
> [ ]

### E2. 文本模式输入
不传截图也能用：直接粘贴一段让你焦虑的文字（聊天记录、邮件、帖子），AI 照样分析。降低使用门槛。
> [ ]

### E3. 分析结果导出
单次分析结果支持导出为图片卡片（精美排版，可保存或分享）或 Markdown 文本。年度报告也可导出 PDF。
> [ ]

### E4. 收藏夹
用户可收藏百宝箱条目、历史分析记录、笔记。收藏夹支持自定义分组（如"反复看的""对我有用""待尝试"）。
> [ ]

### E5. 快捷指令 / Siri 集成
iOS 快捷指令支持：说"记录一条焦虑"即可打开 app 并跳转到上传页。后续可扩展到"开始呼吸练习"等。
> [ ]

### E6. 通知中心优化
用户可精细控制通知类型（每日问候/知识推荐/打卡提醒/周报/新功能），各自独立开关 + 免打扰时段设置。默认非侵入式。
> [ ]

### E7. 多语言支持
系统文案支持中文/英文切换（先用 AI 翻译一轮再人工校对），方便非中文母语用户或海外留学生使用。
> [ ]

### E8. 桌面小组件
iOS/Android 桌面小组件：每日名言、焦虑值折线图、快捷入口（上传/呼吸/笔记）。不需要打开 app 就能获得一点安抚。
> [ ]

## 📚 分类 F：知识库增强（5 条）

### F1. 知识条目朗读版
每条知识支持 TTS 朗读（使用浏览器内置 SpeechSynthesis，无需后端），可调语速。适合不想盯着屏幕看的场景。
> [ ]

### F2. 知识专题合集
管理员可将多条知识打包成"专题"（如"职场焦虑 7 日计划""社交恐惧入门""睡眠改善指南"），用户按顺序阅读，每天解锁一篇。
> [ ]

### F3. 用户贡献知识
用户可提交自己的应对经验或心理学知识，管理员审核后纳入百宝箱，标注"来自用户贡献"。通过审核的用户获得贡献者徽章。
> [ ]

### F4. 知识卡片分享
每条知识条目生成一张精美的分享卡片（带项目 logo 和二维码），用户可保存或转发给朋友。卡片底部带"来自 精神避难所"。
> [ ]

### F5. 个性化知识推荐
根据用户历史焦虑标签和已阅读条目，用简单的协同过滤推荐"你可能需要的知识"。不同于首页的随机推荐——基于用户画像做关联。
> [ ]

## 🛡️ 分类 G：安全与信任（5 条）

### G1. 假登录模式
在登录页设置一个"访客浏览"模式，不注册也能浏览百宝箱和首页，但不能使用分析/笔记等个人功能。降低首次使用心理门槛。
> [ ]

### G2. 紧急退出
App 内任意页面，摇一摇手机或快速点击 3 次顶部标题栏 → 立即跳转到一个无害页面（如天气页面伪装），同时清除当前页面痕迹。防止被人看到敏感内容。
> [ ]

### G3. 数据导出与删除
用户可在设置中一键导出全部个人数据（JSON/CSV），或一键彻底删除所有数据。符合 GDPR/个人信息保护法要求。
> [ ]

### G4. 应用锁
独立的 6 位数字密码或生物识别（Face ID / 指纹）锁住 app 入口，区别于手机系统锁屏。打开 app 时先验证，防止他人借用手机时看到历史分析。
> [ ]

### G5. 敏感内容标记
用户在上传或输入文字时可主动标记"包含极度敏感内容"，该条分析不存日志、处理后立即清除原文（仅保留脱敏后的分析结论）。适用于极端私密场景。
> [ ]

## 🎨 分类 H：视觉与体验（4 条）

### H1. 主题切换
提供 4-6 套配色方案（暖阳/森林/海洋/暗夜/樱花），全局切换。配合系统深色模式自动适配。不影响功能，纯粹视觉情绪调节。
> [ ]

### H2. 字体大小调节
设置中提供 3 档字体大小，适配不同阅读习惯和视力需求。对百宝箱和笔记等大量文字页面尤为重要。
> [ ]

### H3. 加载动画疗愈化
把全局 loading 替换成舒缓的小动画——蒲公英飘散、水波纹扩散、星星闪烁——替代传统的转圈圈。等待分析时本身就是一种安抚。
> [ ]

### H4. 个性化首页问候
根据时间段 + 天气 + 用户历史情绪自动调整首页问候语。比如"今天下雨，你上次说雨天容易低落，要不要试试这个呼吸练习？"需要获取天气 API。
> [ ]
