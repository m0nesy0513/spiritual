## [0.4.0] — 2026-06-20

### 新增 — 搜索系统 + 管理员后台 + 部署上线

**搜索系统**
- 统一搜索 API：并行搜索历史记录 + 百宝箱 + 个人笔记
- `/search` 结果页：Tab 切换（全部/历史/百宝箱/笔记），中文输入法兼容
- 首页搜索框 → 进入搜索页，简约风格

**管理员后台**
- `/admin` 后台页面：三大 Tab（百宝箱管理 / 首页内容 / 管理员 / 用户反馈）
- 知识库 CRUD：新建/编辑/删除，分类/标签/首页推荐管理
- 首页名言/好歌管理：增删改、启用停用
- 系统文案编辑：免责声明/产品说明/问候语等 5 项
- 管理员管理：通过手机号/邮箱添加移除管理员
- 用户反馈查看：管理员列表
- 用户 → 我的 → 管理后台入口；用户反馈提交页 `/suggestions`

**首页种子数据**
- 名言 100 条 + 好歌 100 首，每日随机展示
- 首页随机知识推荐：34 条推荐池每日 3 条

**域名与部署**
- 阿里云 ECS 成都（8.137.78.200）+ Nginx + PM2 部署
- 阿里云 RDS 成都，34 张表
- 域名 spiritualrefuge.xyz（A → 8.137.78.200）
- 本地文件存储 + base64 fallback，适配云环境

### 修复
- Cookie Secure 标记 → `false`（HTTP 环境下设置 Cookie 失败导致无法登录）
- AI prompt 禁止 Markdown 格式化符号
- 分析结果前端清理 ** 
- 反馈页查看模式布局优化
- 百宝箱分页 RAND() → 固定排序修复 key 冲突
- 百宝箱分类栏桌面箭头 + 手机滑动
- 知识摘要优先适用场景

**Coming Soon / 功能入口页**
- `/coming-soon` 页面支持 `?feature=` 参数：4 个待上线功能各自展示独立信息（图标/名称/描述/开发状态）
- 新增返回按钮 + 首页按钮，符合交互规范
- Feature metadata 配置化：`src/config/features.ts`
- 功能入口页卡片 emoji 修复 + feature key 传递
- 修复 `useSearchParams` SSR Suspense 边界问题

## [0.3.0] — 2026-06-17

**首页推荐**
